LWIt is a free app that makes it easy to get fresh new wallpapers on your phone or tablet on a daily basis.
It's a plugin for Muzei, which is an Android live wallpaper app, by Roman Nurik and Ian Lake.
Since Reddit changed API access rules we need a Reddit API auth for each user.
